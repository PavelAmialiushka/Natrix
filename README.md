# Natrix
Pipe drawing tool

For more information please visit https://natrixlabs.ru

# How to build

## Requesites
To build Natrix you need 
1. Microsoft Visual Studio 2019
2. Qt 5.15.2 installed
3. Python3 installed
4. CMake installed
5. Conan installed
6. Makensis installed (if you need to create installer)

## Build
1. Run 
> build.py
